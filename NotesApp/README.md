# How to build

1) Clone the repository

```bash
$ git clone https://github.com/rohit25-1/iOS-Projects.git
```

2) Go to directory

```bash
$ cd NotesApp
```

3) Open in Xcode

4) Run The Server From Notes-API Folder

```bash
node index.js
```

5) Compile and run the app in your simulator


# Requirements

* Xcode
* Node
* MongoDB
